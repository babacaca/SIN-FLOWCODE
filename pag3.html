Le langage C

Le C est un langage de programmation impératif et généraliste. Inventé au début des années 1970 pour réécrire UNIX, C est devenu un des langages les plus utilisés. De nombreux languages plus modernes comme C++, Java et PHP reprennent des aspects de C.
